# School Grades CRUD System

Sistema fullstack de gestión escolar desarrollado con Angular, Node.js, Express y MongoDB.

Este proyecto permite administrar estudiantes y notas escolares mediante una API REST conectada a un frontend moderno desarrollado en Angular.

---

# Características

## Autenticación
- Registro de usuarios
- Inicio de sesión
- JWT Authentication
- Persistencia de sesión mediante token

---

## Gestión de estudiantes
- Crear estudiantes
- Consultar estudiantes
- Eliminar estudiantes

---

## Gestión de notas
- Crear notas
- Consultar notas
- Eliminar notas
- Relación entre estudiantes y notas

---

## Dashboard
- Total de estudiantes
- Total de notas
- Promedio general

---
